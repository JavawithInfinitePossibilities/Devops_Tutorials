    6.  Create a java project
        Commend :
        mvn archetype:generate -DarchetypeGroupId=org.codehaus.mojo.archetypes -DarchetypeArtifactId=pom-root -DarchetypeVersion=RELEASE -DinteractiveMode=false -DartifactId=JavaCentralRepo -Dversion=1.0.0-SNAPSHOT -DgroupId=com.tutorials.java.core -Dpackage=JavaCentralRepo
        ----------------------------------------------------------------------------------------
        archetype:generate : Maven goal
        -DXXX : parameters passed to project
        ----------------------------------------------------------------------------------------
        mvn archetype:generate -DarchetypeGroupId=org.apache.maven.archetypes  -DarchetypeArtifactId=maven-archetype-webapp -DarchetypeVersion=RELEASE -DinteractiveMode=false -DgroupId=com.tutorials.java.core.repo -DartifactId=Java-Core-Repo -Dversion=0.0.1-SNAPSHOT -Dpackage=Java-Core-Repo

    12. Repositories
    13. Create a java project and push it to github.
        mvn archetype:generate -DgroupId=com.bharath -DartifactId=java-project -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
            In windows system to check the structure: E:/Repositories/Devops_Java_project/java-project>tree 
            git init
            git stage .
            Note: Incase any error come in: 
            git config --global --add safe.directory E:/Repositories/Devops_Java_project/java-project 
            git status
            git commit -m "first commit"
            git remote add origin https://github.com/JavawithInfinitePossibilities/devops_java_repository.git
            git remote -v
            git branch
            Note: Use the branch name to push into repository
            git push -u origin master
